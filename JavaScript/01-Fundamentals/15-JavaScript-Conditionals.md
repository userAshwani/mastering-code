## Quick Summary

- Conditional statements choose what code should run.
- `if` runs code when a condition is true.
- `else` runs code when the same condition is false.
- `else if` tests another condition.
- `switch` checks many possible cases.

---

## Key Points

| Statement | Simple Meaning |
| :--- | :--- |
| `if` | Runs code if a condition is true. |
| `else` | Runs code if the condition is false. |
| `else if` | Tests a new condition. |
| `switch` | Runs one of many code blocks. |
| `? :` | Short form of `if...else`. |

## Simple Example

<details open>
<summary><strong>💻 Click to Hide/Show Code Example</strong></summary>
<br>

```javascript
if (condition) {
  // code to execute if the condition is true
}
```
</details>

<details open>
<summary><strong>💻 Click to Hide/Show Code Example</strong></summary>
<br>

```javascript
if (condition) {
  // code to execute if the condition is true
} else {
  // code to execute if the condition is false
}
```
</details>

<details open>
<summary><strong>💻 Click to Hide/Show Code Example</strong></summary>
<br>

```javascript
switch(expression) {
  case x:
    // code block
    break;
  case y:
    // code block
    break;
  default:
    // code block
}
```
</details>

---

## Remember

- Conditions are based on true or false.
- Use `if` for one condition.
- Use `else if` for another condition.
- Use `switch` for many alternatives.
- Use ternary as a short `if...else`.
