## Quick Summary

- `innerHTML` writes into an HTML element.
- `document.write()` writes directly into the page.
- `alert()` shows a popup message.
- `console.log()` writes into the browser console.
- `window.print()` opens the print dialog.

---

## Key Points

| Output Method | Simple Use |
| :--- | :--- |
| `innerHTML` | Display content inside an element. |
| `document.write()` | Write directly to the HTML page. |
| `window.alert()` | Show an alert box. |
| `console.log()` | Show output in the console. |
| `window.print()` | Print the current page. |

## Remember

- Use `innerHTML` to show results on a page.
- Use `console.log()` for testing.
- `document.write()` can replace the page after loading.
- `window.print()` prints the current window.
