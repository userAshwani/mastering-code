## Quick Summary

- A JavaScript program is a list of statements.
- Statements are executed one by one.
- Semicolons separate JavaScript statements.
- Code blocks group statements inside curly braces.
- Keywords tell JavaScript what action to perform.

---

## Key Points

| Item | Simple Meaning |
| :--- | :--- |
| Statement | A command for JavaScript to execute. |
| Semicolon | Separates statements. |
| Whitespace | Makes code easier to read. |
| Code block | Groups statements inside `{}`. |
| Keyword | A reserved word with a special action. |

## Remember

- Statements usually run from top to bottom.
- Semicolons are recommended.
- Extra spaces are ignored by JavaScript.
- Code blocks are used in functions and control structures.
