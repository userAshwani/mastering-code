</div>

---

## Quick Summary

- JavaScript has strings, numbers, BigInt, booleans, undefined, null, symbols, and objects.
- Strings store text.
- Numbers store numeric values.
- Objects store named values.
- Arrays are a type of object.

---

## Key Points

| Type | Simple Example |
| :--- | :--- |
| String | `"Hello"` |
| Number | `42` |
| BigInt | `12345678901234567890n` |
| Boolean | `true` or `false` |
| Undefined | A variable without a value. |
| Null | An empty value. |
| Object | `{ firstName: "John" }` |
| Array | `["Saab", "Volvo", "BMW"]` |

## Remember

- JavaScript variables can hold different data types.
- A string is written inside quotes.
- Numbers can use decimals.
- Arrays and objects are both objects.
- `typeof` checks the type of a value.
