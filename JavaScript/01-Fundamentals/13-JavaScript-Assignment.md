## Quick Summary

- `=` assigns a value to a variable.
- `+=` adds a value to a variable.
- `-=` subtracts a value from a variable.
- `*=` multiplies a variable.
- `/=` divides a variable.

---

## Key Points

| Operator | Same As |
| :--- | :--- |
| `x = y` | `x = y` |
| `x += y` | `x = x + y` |
| `x -= y` | `x = x - y` |
| `x *= y` | `x = x * y` |
| `x **= y` | `x = x ** y` |
| `x /= y` | `x = x / y` |
| `x %= y` | `x = x % y` |

## Remember

- Assignment operators update variable values.
- `+=` can add numbers or join strings.
- `%=` assigns a remainder.
- `&&=`, `||=`, and `??=` are logical assignment operators.
- The spread operator `...` splits iterables into individual elements.
