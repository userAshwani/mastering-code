## Quick Summary

- JavaScript can change HTML content.
- JavaScript can change HTML attributes.
- JavaScript can change CSS styles.
- JavaScript can hide or show HTML elements.

---

## Key Points

| Feature | Simple Meaning |
| :--- | :--- |
| Change content | Update text or HTML inside an element. |
| Change attributes | Change values like `src`, `href`, or `alt`. |
| Change styles | Update CSS properties with JavaScript. |
| Hide elements | Set an element's display to `none`. |
| Show elements | Set an element's display back to visible. |

---

## Remember

- JavaScript works with HTML elements.
- `getElementById()` selects an element by ID.
- `innerHTML` changes element content.
- JavaScript can update both content and design.
