---

## Quick Summary

- Variables can be declared with `var`, `let`, or `const`.
- `let` and `const` are used in modern JavaScript.
- The `=` sign assigns a value.
- Variables can store numbers, strings, and other data.
- Variable names are called identifiers.

---

## Key Points

| Item | Simple Meaning |
| :--- | :--- |
| `var` | Older way to declare variables. |
| `let` | Declares a variable that can change. |
| `const` | Declares a value that should not be reassigned. |
| `=` | Assigns a value. |
| Identifier | The name of a variable. |

---

## Simple Example

```javascript

let y = 6;
let z = x + y;

console.log(z);
```
</details>

<details open>
<summary><strong>💻 Click to Hide/Show Code Example</strong></summary>
<br>

```javascript
const price1 = 5;
const price2 = 6;
let total = price1 + price2;

console.log(total);
```
</details>

---

## Remember

- Use `const` when the value should not change.
- Use `let` when the value can change.
- Use meaningful variable names.
- JavaScript treats `$` and `_` as valid name characters.
