## Quick Summary

- JavaScript has values, variables, operators, expressions, keywords, and comments.
- Fixed values are called literals.
- Variable values are stored in variables.
- JavaScript identifiers are case-sensitive.
- JavaScript commonly uses lower camel case.

---

## Key Points

| Part | Simple Meaning |
| :--- | :--- |
| Literals | Fixed values like `10.50` or `"John Doe"`. |
| Variables | Names that store values. |
| Operators | Symbols like `+`, `-`, `*`, and `=`. |
| Expressions | Code that computes a value. |
| Keywords | Reserved words like `let`, `const`, and `var`. |

## Remember

- Strings are written inside quotes.
- Numbers can be written with or without decimals.
- `=` assigns values.
- `lastName` and `lastname` are different names.
