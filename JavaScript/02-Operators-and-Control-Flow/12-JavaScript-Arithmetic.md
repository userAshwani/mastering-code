## Quick Summary

- Arithmetic operators work with numbers.
- `+`, `-`, `*`, and `/` perform basic math.
- `%` returns the division remainder.
- `++` increases a number by 1.
- `--` decreases a number by 1.

---

## Key Points

| Operator | Simple Meaning |
| :--- | :--- |
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `%` | Modulus, or remainder |
| `++` | Increment |
| `--` | Decrement |
| `**` | Exponentiation |

## Remember

- Operands are the numbers used in an operation.
- Operators define the math action.
- Multiplication and division run before addition and subtraction.
- Parentheses can change the calculation order.
- Operations with the same precedence run from left to right.
